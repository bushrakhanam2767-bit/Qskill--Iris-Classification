# 🌸 Iris Flower Classification & Analysis Dashboard

An advanced, interactive Machine Learning web application designed to classify Iris flower species (*Setosa, Versicolor, Virginica*) based on their morphological measurements. Built with **Python, Scikit-Learn, and Streamlit**.

## 🚀 Features
- **Real-Time Classification:** Adjust morphological features using sidebar sliders to instantly predict the species.
- **Confidence Metrics:** Displays precision breakdown and probability tables for the model's predictions.
- **Dynamic Visual Evaluation:** Renders live field samples (images) and high-resolution scatter plots mapping user coordinates.
- **Clean Production UI:** Fully structured in standard corporate English.

## 🛠️ Tech Stack & Libraries
- **Language:** Python
- **Framework:** Streamlit (Web UI)
- **Machine Learning:** Scikit-Learn (K-Nearest Neighbors Classifier)
- **Data Analytics:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborns
